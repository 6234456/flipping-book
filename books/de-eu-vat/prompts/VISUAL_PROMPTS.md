# VAT Atlas 视觉生成提示词规范 v0.3

## 全局风格

竖版单页 atlas / magazine page；蓝白写实杂志图册风格；轻微纸张质感；德国 / 欧盟地图淡背景；橄榄枝、纸夹、票据、印章、法规书、税务文件等写实装饰。中文为主，德语只作小标签或 tooltip 锚点。不要使用横版 PPT / corporate handout / flat vector dashboard 风格。

## 生图内容规则

- 不允许图像模型自由编造税务判断。
- 使用 `copy/page-copy/*.md` 中的短版文案作为文字层。
- 复杂说明放 tooltip 或页边补充，不塞进主图。
- 每页保留：核心问题、判断流程、关键规则、高风险误区、本页术语。

## 禁止项

- 不得把住宿 + 早餐作为从给付正面例子。
- 不得使用“工程供货 / 工程服务”。
- 不得使用“为他人账户”。
- 不得使用“第一供应方 / 最终取得方”。
- 不得写“B2B 一律 reverse charge”。
- 不得写“EXW 自动免税”。
- 不得把 PDF 写成电子发票。
