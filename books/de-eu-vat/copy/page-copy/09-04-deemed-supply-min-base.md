# 09-04 无偿价值转移与最低计税基础

**副标题：** 免费不一定无 VAT；低价也不一定按低价计税

## 核心问题 / 页面定位

区分完全无偿与有偿但低价：前者检查视同给付，后者检查最低计税基础。

## 判断流程

1. 是否存在对价？
2. 无偿：货物、服务还是资产使用？
3. 货物：是否企业资产且满足进项税 gate？
4. 是否有小额礼品、样品或业务原因例外？
5. 低价：是否为员工、股东或关联方场景？
6. §10(4) value 是否高于实际对价，并受 market value cap 限制？

## 关键规则

- 员工长期私人使用公司车通常不是无偿，而是有偿服务给付。
- 关联方低价交易不是简单“按市价计税”。
- recipient 不能因为赠与方内部计提 VAT 就抵扣进项税。

## 高风险误区

- 免费福利不能只做工资税处理。
- ERP gift item 不能自动排除 VAT。

## 本页术语

- 无偿价值转移
- 最低计税基础
- 计税基础
- 对价
- 进项税抵扣

## 法规 / 开发锚点

- UStG §3 Abs.1b
- UStG §3 Abs.9a
- UStG §10 Abs.4/5
