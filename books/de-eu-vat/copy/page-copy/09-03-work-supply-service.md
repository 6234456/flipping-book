# 09-03 加工供货与加工服务边界

**副标题：** Werklieferung / Werkleistung 不限于建筑工程

## 核心问题 / 页面定位

当加工方对客户或第三方物品加工、维修、安装或改造，并使用自购材料时，需要判断是加工供货还是加工服务。

## 判断流程

1. 是否对他人物品加工 / 处理？
2. 是否使用自购材料？
3. 材料是否只是辅料 / 附属材料？
4. 材料是否构成主要材料？
5. 材料是否包含在加工成果中？
6. 动产维修仍不明确时，才检查 50% 材料比例线索。

## 关键规则

- 材料 / 人工金额比例不是首要判断标准。
- 材料费与人工费分列，不自动拆分 VAT treatment。
- 供应商销售自有产品通常先按普通货物供应判断。

## 高风险误区

- 禁用“工程供货 / 工程服务”。
- 不要用“材料价值 > 人工价值”直接下结论。

## 本页术语

- 加工供货
- 加工服务
- 加工成果
- 自购材料
- 辅料 / 附属材料

## 法规 / 开发锚点

- UStG §3 Abs.4
- UStAE 3.8
