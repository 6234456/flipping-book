# SC-02 欧盟货物交易

**副标题：** 销售、购置、自有调拨，还是寄售库存？

## 核心问题 / 页面定位

第一步不是看发票，而是看货物是否实际跨越欧盟成员国边界。

## 判断流程

1. 先确认是货物，不是服务。
2. 确认货物是否实际跨越欧盟成员国边界。
3. 有客户销售：检查欧盟内免税供应 / 欧盟内购置。
4. 无客户销售：检查自有货物调拨。
5. 仓库后续提货：检查寄售库存规则。
6. 多方买卖一次运输：转入连环交易。

## 关键规则

- VAT ID 不是运输证明。
- 自有货物跨境移动也可能产生 VAT 后果。
- Intrastat 是统计申报，不是 VAT 申报。

## 高风险误区

- EXW 自提不是免税保证。
- 寄售库存必须满足 §6b 条件。

## 本页术语

- 欧盟内免税供应
- 欧盟内购置
- 欧盟内自有货物调拨
- 寄售库存
- 到达证明

## 法规 / 开发锚点

- UStG §1a
- UStG §6a
- UStG §6b
- UStG §18a
