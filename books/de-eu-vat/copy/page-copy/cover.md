# 00 德国 / 欧盟 VAT 财务速查图册

**副标题：** 常用 B2B 场景 · 法规提示 · 可点击 Drill-down 导览

## 核心问题 / 页面定位

一本面向财务团队的德国 / 欧盟 VAT 可交互速查图册。视觉页为入口，术语、发票、申报、证据与风险提示由结构化数据层控制。

## 判断流程

1. 先判断交易性质
2. 再判断给付地点
3. 再判断纳税义务人
4. 最后落到发票、申报和证据

## 关键规则

- 中文主标签优先，德语作为法规锚点。
- 图像只承载视觉；专业文案以结构化 text layer 为准。
- 热点、术语、notes、comments 均由 manifest 和 registry 驱动。

## 高风险误区

- 不要把图像中的生成文字作为税法最终事实。
- 图片版本变化必须同步 overlay 与 comment anchor。
