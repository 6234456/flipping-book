# G 术语表与悬浮提示词库

**副标题：** 中文主译名、德语锚点、tooltip 统一控制

## 核心问题 / 页面定位

术语表不是普通附录，而是全书术语控制层。图中文字、tooltip、页边补充、法规锚点和最终打包文件必须一致。

## 判断流程

1. 中文主译名优先。
2. 首次出现：中文术语（德语原文）。
3. 后续出现：只写中文主译名。
4. 图中放短定义，长解释进入 tooltip。
5. 同一术语全书只允许一个中文译法。

## 关键规则

- Werklieferung = 加工供货。
- Werkleistung = 加工服务。
- für fremde Rechnung = 内部法律后果归属于委托方。
- E-Rechnung = 电子发票。
- 三角贸易主图：第一卖方 / 中间商 / 最终买方。

## 高风险误区

- 禁用：工程供货 / 工程服务。
- 禁用：为他人账户。
- 禁用：第一供应方 / 最终取得方。
