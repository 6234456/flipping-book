# 09 交易性质判断

**副标题：** 统一给付、主从给付、加工供货、赔偿与委托买卖

## 核心问题 / 页面定位

很多 VAT 错误不是发生在申报字段，而是发生在更早一步：交易本身到底是什么。

## 判断流程

1. 一个给付还是多个给付？
2. 货物供应还是服务给付？
3. 是否存在法定拟制？
4. 是否存在法定拆分要求？
5. 是否只是赔偿或价款调整？
6. 是否是委托买卖 / 行纪结构？

## 关键规则

- 价值比例不是决定性因素。
- 发票分行不是 VAT classification。
- 专业术语首次出现：中文译名 + 德语原文。

## 高风险误区

- 把工程 / 加工 / 维修全部误判为普通服务。
- 把合同 package 误认为统一给付。

## 本页术语

- 统一给付
- 主给付
- 从给付
- 加工供货
- 加工服务
