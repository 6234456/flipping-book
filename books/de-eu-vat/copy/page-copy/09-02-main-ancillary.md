# 09-02 主给付 / 从给付

**副标题：** 该组成部分是否对客户具有独立经济目的？

## 核心问题 / 页面定位

从给付通常跟随主给付，但这只是一般原则；存在法定拆分要求时，必须拆分处理。

## 判断流程

1. 是否已经识别一个主给付？
2. 该组成部分是否对客户有独立经济目的？
3. 是否只是为了更好取得主给付？
4. 是否通常伴随主交易出现？
5. 是否可单独购买、取消或替换？
6. 是否存在法定拆分要求？

## 关键规则

- 单独收费本身不是决定性因素。
- 住宿 + 早餐 / 停车 / minibar / wellness 不能作为从给付正面例子。
- 销售包装与周转运输器具需区分。

## 高风险误区

- “从给付跟随主给付”不是 universal rule。
- 安装、培训、维护、保证承诺不能机械作为销售从给付。

## 本页术语

- 主给付
- 从给付
- 独立给付
- 法定拆分要求
- 住宿服务

## 法规 / 开发锚点

- UStAE 3.10 Abs.5
- UStAE 12.16
