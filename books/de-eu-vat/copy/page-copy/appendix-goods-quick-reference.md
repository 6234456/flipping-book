# A 常用货物场景速查

**副标题：** 欧盟客户、第三国客户、自提与 DDP 风险

## 核心问题 / 页面定位

先判断货物路径，再决定进入哪个章节。客户国别和 Incoterms 都不是 VAT 结论。

## 判断流程

1. 德国公司是卖方还是买方？
2. 货物留在德国、去欧盟，还是去第三国？
3. 欧盟方向：检查 VAT ID 与到达证明。
4. 第三国出口：检查出口完成确认。
5. 第三国进口：检查清关主体、进口税债务人和处分权。
6. 多方买卖一次运输：转入连环交易。

## 关键规则

- VAT ID 不是到达证明。
- EXW 自提不是免税保证。
- 商业发票不是出口完成证明。
- DDP 德国交付可能触发供应方德国 VAT 风险。

## 高风险误区

- 客户在欧盟 / 第三国不等于自动免税。
- 清关付款人不一定是进口增值税抵扣权利人。

## 本页术语

- 到达证明

- 进口税债务人
