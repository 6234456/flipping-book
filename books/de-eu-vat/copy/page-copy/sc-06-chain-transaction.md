# SC-06 连环交易

**副标题：** 多方买卖、一次运输，谁享受跨境免税？

## 核心问题 / 页面定位

同一货物、多方连续买卖、一次直接运输时，唯一一次运输只能归属于其中一段供应。

## 判断流程

1. 是否为同一货物连续买卖？
2. 货物是否从第一方直接到最后一方？
3. 谁安排唯一一次运输？
4. 中间商运输时，是否使用起运国 VAT ID / 税号？
5. 确认有运输归属的供应。
6. 其他段作为静止供应判断地点和税务处理。

## 关键规则

- 只有一段供应可以享受运输归属。
- 静止供应不是无税。
- 中间商使用哪个 VAT ID，可能改变整条链。

## 高风险误区

- 不要把所有三方交易写成三角贸易。
- Incoterms 不能替代运输归属判断。

## 本页术语

- 连环交易
- 有运输归属的供应
- 静止供应
- 中间商
- 起运成员国

## 法规 / 开发锚点

- UStG §3 Abs.6a
- UStAE 3.14
