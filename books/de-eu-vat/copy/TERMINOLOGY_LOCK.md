# VAT Atlas 术语锁定清单 v0.3

## 全局原则

- 中文主译名优先；德语原文仅作为法规锚点、tooltip 或括号说明。
- 图中文字不得中英文混杂；英文/德语只作为术语标签。
- 图像生成文字不得作为专业内容唯一来源；以 `data/glossary.json`、`copy/page-copy/*.md` 和 `data/contents.json` 为准。

## 固定译名

| 原文 | 中文主译名 | 禁用 / 注意 |
|---|---|---|
| Werklieferung | 加工供货 | 禁用“工程供货” |
| Werkleistung | 加工服务 | 禁用“工程服务” |
| Werk | 加工成果 | 不译成“工程” |
| für fremde Rechnung | 内部法律后果归属于委托方 | 禁用“为他人账户” |
| E-Rechnung | 电子发票 | 不作为中文主标题保留 E-Rechnung |
| erster Lieferer | 第一卖方 | 图中禁用“第一供应方” |
| erster Abnehmer / mittlerer Unternehmer | 中间商（第一买方 / 第二卖方） | 视版面可简写为“中间商” |
| letzter Abnehmer | 最终买方 | 图中禁用“最终取得方” |
| Zusammenfassende Meldung | 欧盟交易汇总申报 | 可简称 ZM |
| Intrastat | 欧盟内货物流统计 | 不是 VAT 申报 |

## Tooltip 固定解释：für fremde Rechnung

受托买卖人对外以自己名义与第三人交易，并直接对第三人享有权利、承担义务；但在其与委托方的内部关系中，交易利益、成本、结算义务、风险和最终法律后果归属于委托方。
