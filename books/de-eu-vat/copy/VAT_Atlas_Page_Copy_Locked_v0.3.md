# 01 VAT 判断总框架

**副标题：** 不要从税率或发票 wording 开始

## 核心问题 / 页面定位

VAT 判断的错误往往发生在第一步：交易本身到底是什么。必须先判断交易性质，再进入地点、纳税义务、发票、申报和证据。

## 判断流程

1. 交易性质：供应了什么？
2. 给付地点：在哪里征税？
3. 纳税义务：谁申报 VAT？
4. 发票处理：是否列税或提示反向征税？
5. 申报路径：UStVA、ZM、Intrastat？
6. 证据链：VAT ID、到达证明、出口证明、进口凭证？

## 关键规则

- 合同标题不是 VAT 分类。
- Incoterms 不是 VAT 结论。
- ERP 税码是结果，不是判断依据。

## 高风险误区

- 先看发票税率，会掩盖交易性质错误。
- 把服务、货物、加工供货混同会连锁影响地点与申报。

## 本页术语

- 给付
- 给付地点
- 纳税义务人
- 证据链

## 法规 / 开发锚点

- UStG §1
- UStG §3
- UStG §13b
- UStG §14

# 09 交易性质判断

**副标题：** 统一给付、主从给付、加工供货、赔偿与委托买卖

## 核心问题 / 页面定位

很多 VAT 错误不是发生在申报字段，而是发生在更早一步：交易本身到底是什么。

## 判断流程

1. 一个给付还是多个给付？
2. 货物供应还是服务给付？
3. 是否存在法定拟制？
4. 是否存在法定拆分要求？
5. 是否只是赔偿或价款调整？
6. 是否是委托买卖 / 行纪结构？

## 关键规则

- 价值比例不是决定性因素。
- 发票分行不是 VAT classification。
- 专业术语首次出现：中文译名 + 德语原文。

## 高风险误区

- 把工程 / 加工 / 维修全部误判为普通服务。
- 把合同 package 误认为统一给付。

## 本页术语

- 统一给付
- 主给付
- 从给付
- 加工供货
- 加工服务

## 法规 / 开发锚点

- UStAE 3.10
- UStAE 3.8
- UStAE 1.3

# 09-01 统一给付 / 单一交易

**副标题：** 客户取得一个整体经济结果，还是多个独立给付？

## 核心问题 / 页面定位

判断多个组成部分是否应作为一个 VAT supply 处理，还是拆成多个 taxable components。

## 判断流程

1. 是否存在多个组成部分？
2. 是否由同一企业提供？
3. 客户是否追求一个整体经济结果？
4. 是否存在主给付与从给付？
5. 多个元素是否紧密结合、拆分显得人为？
6. 是否存在法定拆分要求？

## 关键规则

- 合同、发票、价格结构只是 evidence。
- 统一经济交易不得人为拆分。
- 多个独立主给付也不能因 package 强行合并。

## 高风险误区

- 金额占比不能替代普通消费者视角。
- 发票分行不等于多个独立给付。
- 法定拆分要求优先。

## 本页术语

- 统一给付
- 多个独立给付
- 普通消费者视角
- 法定拆分要求

## 法规 / 开发锚点

- UStAE 3.10
- EuGH C-349/96 CPP
- EuGH C-41/04 Levob

# 09-02 主给付 / 从给付

**副标题：** 该组成部分是否对客户具有独立经济目的？

## 核心问题 / 页面定位

从给付通常跟随主给付，但这只是一般原则；存在法定拆分要求时，必须拆分处理。

## 判断流程

1. 是否已经识别一个主给付？
2. 该组成部分是否对客户有独立经济目的？
3. 是否只是为了更好取得主给付？
4. 是否通常伴随主交易出现？
5. 是否可单独购买、取消或替换？
6. 是否存在法定拆分要求？

## 关键规则

- 单独收费本身不是决定性因素。
- 住宿 + 早餐 / 停车 / minibar / wellness 不能作为从给付正面例子。
- 销售包装与周转运输器具需区分。

## 高风险误区

- “从给付跟随主给付”不是 universal rule。
- 安装、培训、维护、保证承诺不能机械作为销售从给付。

## 本页术语

- 主给付
- 从给付
- 独立给付
- 法定拆分要求
- 住宿服务

## 法规 / 开发锚点

- UStAE 3.10 Abs.5
- UStAE 12.16

---

# 09-03 加工供货与加工服务边界

**副标题：** Werklieferung / Werkleistung 不限于建筑工程

## 核心问题 / 页面定位

当加工方对客户或第三方物品加工、维修、安装或改造，并使用自购材料时，需要判断是加工供货还是加工服务。

## 判断流程

1. 是否对他人物品加工 / 处理？
2. 是否使用自购材料？
3. 材料是否只是辅料 / 附属材料？
4. 材料是否构成主要材料？
5. 材料是否包含在加工成果中？
6. 动产维修仍不明确时，才检查 50% 材料比例线索。

## 关键规则

- 材料 / 人工金额比例不是首要判断标准。
- 材料费与人工费分列，不自动拆分 VAT treatment。
- 供应商销售自有产品通常先按普通货物供应判断。

## 高风险误区

- 禁用“工程供货 / 工程服务”。
- 不要用“材料价值 > 人工价值”直接下结论。

## 本页术语

- 加工供货
- 加工服务
- 加工成果
- 自购材料
- 辅料 / 附属材料

## 法规 / 开发锚点

- UStG §3 Abs.4
- UStAE 3.8

# 09-04 无偿价值转移与最低计税基础

**副标题：** 免费不一定无 VAT；低价也不一定按低价计税

## 核心问题 / 页面定位

区分完全无偿与有偿但低价：前者检查视同给付，后者检查最低计税基础。

## 判断流程

1. 是否存在对价？
2. 无偿：货物、服务还是资产使用？
3. 货物：是否企业资产且满足进项税 gate？
4. 是否有小额礼品、样品或业务原因例外？
5. 低价：是否为员工、股东或关联方场景？
6. §10(4) value 是否高于实际对价，并受 market value cap 限制？

## 关键规则

- 员工长期私人使用公司车通常不是无偿，而是有偿服务给付。
- 关联方低价交易不是简单“按市价计税”。
- recipient 不能因为赠与方内部计提 VAT 就抵扣进项税。

## 高风险误区

- 免费福利不能只做工资税处理。
- ERP gift item 不能自动排除 VAT。

## 本页术语

- 无偿价值转移
- 最低计税基础
- 计税基础
- 对价
- 进项税抵扣

## 法规 / 开发锚点

- UStG §3 Abs.1b
- UStG §3 Abs.9a
- UStG §10 Abs.4/5

# 09-05 损害赔偿还是对价

**副标题：** 合同写“赔偿”不等于 VAT 上非应税

## 核心问题 / 页面定位

核心是是否存在给付交换：付款方是否取得可识别的给付、容忍、放弃、使用权或经济利益。

## 判断流程

1. 是否存在给付交换？
2. 是否只是补偿已发生损害？
3. 是否对应容忍 / 放弃 / 不作为？
4. 是否只是调整原交易价款？
5. 是否涉及提前终止、取消、改约？
6. take-or-pay 是否对应可用性、保留产能或补提权？

## 关键规则

- 真正损害赔偿没有给付交换。
- 质量扣款和折让通常先检查 §17。
- 荷兰 VAT 的 take-or-pay 判断对德国有高参考价值。

## 高风险误区

- take-or-pay 不应默认作为真正损害赔偿。
- 客户未实际提货 / 使用，不等于没有给付交换。

## 本页术语

- 损害赔偿
- 真正损害赔偿
- 对价
- 给付交换
- 最低采购义务

## 法规 / 开发锚点

- UStAE 1.1
- UStAE 1.3
- CJEU C-295/17 MEO
- CJEU C-43/19 Vodafone Portugal

# 09-06 委托买卖交易 / 行纪交易

**副标题：** 以自己名义交易，内部法律后果归属于委托方

## 核心问题 / 页面定位

判断中间人到底只是显名代理，还是在 VAT 上被视为买入再卖出。关键是对外名义与内部法律后果的分层。

## 判断流程

1. 交易对象是货物还是服务？
2. 中间人是否以自己名义对外交易？
3. 内部法律后果是否归属于委托方？
4. 货物：销售委托还是采购委托？
5. 服务：是否形成服务委托拟制？
6. 分别检查发票链、货物流、服务流和 VAT 拟制链。

## 关键规则

- 受托买卖人对外直接享有权利、承担义务。
- 内部利益、成本、结算义务、风险和最终法律后果归属于委托方。
- “为他人账户”为禁用译法。

## 高风险误区

- 只收佣金不代表只是代理。
- 货物流直达客户不代表 VAT 链条跳过中间人。

## 本页术语

- 委托买卖交易
- 委托方
- 受托买卖人
- 内部法律后果归属于委托方
- 法律拟制

## 法规 / 开发锚点

- UStG §3 Abs.3
- UStG §3 Abs.11
- HGB §383

# 09-07 车辆保养、换机油与质保维修

**副标题：** 不要从零件价值开始判断

## 核心问题 / 页面定位

客户取得的是车辆维护 / 修复结果，还是单独购买零部件？本页训练统一给付、主从给付、加工供货 / 加工服务三层判断。

## 判断流程

1. 客户是否另行支付？
2. 是单独零部件，还是维护 / 修复结果？
3. 是否属于统一保养或维修给付？
4. 是否使用维修方自购的主要零部件？
5. 质保场景中，谁实际承担义务，谁向谁结算？

## 关键规则

- 换机油通常先看维护结果，不自动拆分机油和人工。
- 保外更换发动机高风险进入加工供货审查。
- 质保期内更换发动机，客户通常不是重新购买发动机。

## 高风险误区

- 客户免费不代表整条维修结算链没有 VAT。
- 商业延长保修不能与法定瑕疵担保混同。

## 本页术语

- 法定瑕疵担保
- 补救履行
- 商业保证
- 加工供货
- 加工服务

## 法规 / 开发锚点

- UStAE 3.10
- UStAE 3.8
- BGB §437
- BGB §439

# SC-01 德国国内 B2B

**副标题：** 普通应税、免税，还是反向征税？

## 核心问题 / 页面定位

德国供应商 + 德国客户，不等于一定 19%。应按交易性质、给付地点、免税、反向征税、税率和发票顺序判断。

## 判断流程

1. 先完成交易性质判断。
2. 确认给付地点是否在德国。
3. 检查是否免税。
4. 检查是否适用反向征税。
5. 判断税率：19%、7% 或特殊规则。
6. 检查发票、申报和进项税抵扣。

## 关键规则

- B2B 不等于自动 reverse charge。
- 有 VAT 不等于买方一定可抵扣。
- 税码是结果，不是判断依据。

## 高风险误区

- 免税交易误开 VAT 可能触发 §14c。
- 低税率必须有明确法律依据。

## 本页术语

- 德国国内交易
- 应税交易
- 标准税率
- 优惠税率
- 进项税抵扣

## 法规 / 开发锚点

- UStG §1
- UStG §4
- UStG §12
- UStG §13b
- UStG §14c

# SC-02 欧盟货物交易

**副标题：** 销售、购置、自有调拨，还是寄售库存？

## 核心问题 / 页面定位

第一步不是看发票，而是看货物是否实际跨越欧盟成员国边界。

## 判断流程

1. 先确认是货物，不是服务。
2. 确认货物是否实际跨越欧盟成员国边界。
3. 有客户销售：检查欧盟内免税供应 / 欧盟内购置。
4. 无客户销售：检查自有货物调拨。
5. 仓库后续提货：检查寄售库存规则。
6. 多方买卖一次运输：转入连环交易。

## 关键规则

- VAT ID 不是运输证明。
- 自有货物跨境移动也可能产生 VAT 后果。
- Intrastat 是统计申报，不是 VAT 申报。

## 高风险误区

- EXW 自提不是免税保证。
- 寄售库存必须满足 §6b 条件。

## 本页术语

- 欧盟内免税供应
- 欧盟内购置
- 欧盟内自有货物调拨
- 寄售库存
- 到达证明

## 法规 / 开发锚点

- UStG §1a
- UStG §6a
- UStG §6b
- UStG §18a

# SC-03 欧盟 B2B 服务

**副标题：** 客户所在地规则，还是特殊给付地点？

## 核心问题 / 页面定位

跨境服务先判断服务性质、客户身份和固定机构，再检查不动产、活动入场、短期租赁、餐饮等特殊地点规则。

## 判断流程

1. 确认是服务给付。
2. 判断客户是否为企业客户。
3. 判断服务提供给客户总部还是固定机构。
4. 检查是否属于特殊服务。
5. 德国供应商向欧盟企业：通常不列德国 VAT，检查 ZM。
6. 德国企业从欧盟供应商取得服务：通常按 §13b 自申报。

## 关键规则

- VAT ID 不是全部判断。
- 付款方不一定是服务接受方。
- 服务交易不做 Intrastat。

## 高风险误区

- 不动产、活动入场、短期租赁、餐饮不能套普通 B2B 规则。
- 发票写 reverse charge 不等于法律上适用。

## 本页术语

- 服务给付
- 给付地点
- 固定机构
- 服务接受方
- 反向征税

## 法规 / 开发锚点

- UStG §3a
- VAT Directive Art.44
- UStG §13b

# SC-04 §13b 反向征税

**副标题：** 什么时候由接受方申报 VAT？

## 核心问题 / 页面定位

先确认交易在德国应税且不免税，再确认是否落入 §13b 列举场景，最后确认接受方是否满足身份条件。

## 判断流程

1. 交易是否在德国应税？
2. 是否应税且不免税？
3. 是否属于 §13b 列举场景？
4. 接受方是否满足身份条件？
5. 发票是否不列税并注明反向征税？
6. 接受方是否完成自申报与抵扣检查？

## 关键规则

- 境外供应商德国给付：欧盟或第三国供应商均需检查。
- 废旧金属回收替代清洁服务作为典型主卡。
- 建筑服务必须检查接受方是否持续提供建筑服务。

## 高风险误区

- B2B 不等于自动 §13b。
- 德国 VAT ID 不等于德国设立。
- 发票文字不能替代法律判断。

## 本页术语

- 反向征税
- 境外企业
- 建筑服务
- 废旧金属回收
- 证明表格

## 法规 / 开发锚点

- UStG §13b
- UStAE 13b.3
- UStAE 13b.11

# SC-05 进口与出口

**副标题：** 第三国货物流、出口证明与进口增值税

## 核心问题 / 页面定位

第三国客户地址不等于出口免税；清关付款人不一定是进口增值税抵扣权利人。

## 判断流程

1. 先确认是货物。
2. 货物离开欧盟：检查出口供应和出口证明。
3. 货物进入德国：检查进口清关、进口增值税和抵扣资格。
4. 多方买卖一次运输：转入连环交易。
5. Incoterms 只作线索，不是 VAT 结论。
6. 最后检查发票、海关凭证、ERP 税码和申报。

## 关键规则

- Commercial invoice 不是出口完成证明。
- DDP 德国交付可能触发供应方德国 VAT 风险。
- 进口增值税不是供应商发票 VAT。

## 高风险误区

- EXW 自提不是出口免税保证。
- 货代参与清关通常不取得进口 VAT 抵扣资格。

## 本页术语

- 第三国地区
- 出口供应
- 出口完成确认
- 进口增值税
- 处分权

## 法规 / 开发锚点

- UStG §6
- UStG §21
- UStG §15 Abs.1 Nr.2
- UStAE 15.8

# SC-06 连环交易

**副标题：** 多方买卖、一次运输，谁享受跨境免税？

## 核心问题 / 页面定位

同一货物、多方连续买卖、一次直接运输时，唯一一次运输只能归属于其中一段供应。

## 判断流程

1. 是否为同一货物连续买卖？
2. 货物是否从第一方直接到最后一方？
3. 谁安排唯一一次运输？
4. 中间商运输时，是否使用起运国 VAT ID / 税号？
5. 确认有运输归属的供应。
6. 其他段作为静止供应判断地点和税务处理。

## 关键规则

- 只有一段供应可以享受运输归属。
- 静止供应不是无税。
- 中间商使用哪个 VAT ID，可能改变整条链。

## 高风险误区

- 不要把所有三方交易写成三角贸易。
- Incoterms 不能替代运输归属判断。

## 本页术语

- 连环交易
- 有运输归属的供应
- 静止供应
- 中间商
- 起运成员国

## 法规 / 开发锚点

- UStG §3 Abs.6a
- UStAE 3.14

---

# SC-07 三角贸易

**副标题：** 欧盟内三方简化，谁承担目的国 VAT？

## 核心问题 / 页面定位

三角贸易是连环交易中的特殊简化规则；三方以上交易不能整链套用，必须先按连环交易切分。

## 判断流程

1. 是否为同一货物三方连续买卖？
2. 三方是否使用不同成员国 VAT ID？
3. 货物是否从第一卖方直接到最终买方？
4. 运输是否由第一卖方或中间商作为第一买方安排？
5. 中间商是否未在目的国设立，并使用非目的国 VAT ID？
6. 发票是否写明三角贸易和最终买方纳税义务？

## 关键规则

- 主图使用：第一卖方 / 中间商 / 最终买方。
- 最终买方自提通常不能适用简化。
- 第三国公司也可能参与，但必须以欧盟成员国 VAT ID 进入判断。

## 高风险误区

- 三方交易不等于三角贸易。
- 三方以上交易不能整链套用；只有合格连续三方片段可能适用。

## 本页术语

- 三角贸易
- 第一卖方
- 中间商（第一买方 / 第二卖方）
- 最终买方
- 纳税义务转移

## 法规 / 开发锚点

- UStG §25b
- UStG §14a Abs.7
- VAT Directive Art.141

---

# SC-08 发票与申报

**副标题：** 把 VAT 判断落到 invoice、reporting、evidence

## 核心问题 / 页面定位

发票与申报不是 VAT 判断的起点，而是 VAT 判断的结果。

## 判断流程

1. 先确认 VAT 定性。
2. 判断发票应列 VAT、不列 VAT、反向征税、免税还是三角贸易 wording。
3. 判断是否进入 UStVA、ZM、Intrastat。
4. 检查电子发票与结构化数据。
5. 检查到达证明、出口证明、进口凭证。
6. 检查 §14c、§17 与进项税抵扣风险。

## 关键规则

- 电子发票不是普通 PDF。
- UStVA、ZM、Intrastat 是三套不同申报。
- ERP 税码是结果，不是法律依据。

## 高风险误区

- 发票有 VAT 不等于一定可抵扣。
- 错误列税可能触发 §14c。

## 本页术语

- 发票
- 电子发票
- 增值税预申报
- 欧盟交易汇总申报
- 欧盟内货物流统计

## 法规 / 开发锚点

- UStG §14
- UStG §14a
- UStG §14c
- UStG §18a

# A 常用货物场景速查

**副标题：** 欧盟客户、第三国客户、自提与 DDP 风险

## 核心问题 / 页面定位

先判断货物路径，再决定进入哪个章节。客户国别和 Incoterms 都不是 VAT 结论。

## 判断流程

1. 德国公司是卖方还是买方？
2. 货物留在德国、去欧盟，还是去第三国？
3. 欧盟方向：检查 VAT ID 与到达证明。
4. 第三国出口：检查出口完成确认。
5. 第三国进口：检查清关主体、进口税债务人和处分权。
6. 多方买卖一次运输：转入连环交易。

## 关键规则

- VAT ID 不是到达证明。
- EXW 自提不是免税保证。
- 商业发票不是出口完成证明。
- DDP 德国交付可能触发供应方德国 VAT 风险。

## 高风险误区

- 客户在欧盟 / 第三国不等于自动免税。
- 清关付款人不一定是进口增值税抵扣权利人。

## 本页术语

- 客户自提
- 到达证明
- 出口完成确认
- 进口税债务人
- 清关申报人

## 法规 / 开发锚点

- UStG §6a
- UStG §6
- UStG §3 Abs.8
- UStAE 3.14
